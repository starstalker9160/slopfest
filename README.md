# slopfest (chrome extension)
![hero image](./repo%20stuff/hero.jpg)

## Installation
1. Clone the repo
2. Put it somewhere
3. Go to `chrome://extensions`
4. Enable developer mode
5. Click `Load unpacked` and navigate to the slopfest folder
